15.vuejs路由

vue.js路由允许通过不同的url访问不同的内容。

通过vue.js可以实现多视图的单页web应用。

vue.js路由需要载vue-router库

15.1安装

直接下载/CDN

http://unpkg.com/vue-router/dist/vue-router.js

NPM

淘宝镜像：

cnpm install vue-router

15.2简单实例

vue.js+vue-router可以很简单实现单页应用

<router-link>是一个组件，用于设置一个导航链接，切换不同HTML内容。

hello vue-router

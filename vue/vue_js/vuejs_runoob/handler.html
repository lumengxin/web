11.1事件修饰符

vue.js为v-on提供了事件修饰符来处理DOM事件细节。如event.preventDefault()。

vue.js通过点（.）表示的指令后缀来调用修饰符。

  • .stop
  • .prevent
  • .capture
  • .self
  • .once

11.2按键修饰符

vue允许为v-on在监听事件时添加按键修饰符。

< input v-on:keyup.enter="submit">

< -- 缩写 -->

< input @keyup.enter="submit">

常用按键别名：

  • .enter
  • .tab
  • .delete
  • .space

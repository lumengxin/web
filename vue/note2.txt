#14.1钩子
##14.1.1钩子函数
	指令定义函数提供了几个钩子函数：
		bind:只调用一次，指令第一次绑定到该元素时。
		inserted:被绑定元素插入父节点时调用。
		update:更新时。
		comonentUpdataed:一次更新周期。
		unbind:解绑。
##14.1.2钩子函数
	钩子函数有：
		el:指令所绑定的元素，可以用来直接操作DOM。
		binding:一个对象，包含以下属性：name value oldValue...
		vnode:vue编译生成的虚拟节点。

16.vuejs过渡&动画
#16.1过渡
vue在插入，更新或者移除DOM时，提供多种不同方式的应用过渡效果。
vue提供了内置的过渡封装组件，用于包裹要实现过渡效果的组件。
##16.1.1语法格式：
	过渡其实就是一个淡入淡出的效果。Vue在元素显示与隐藏的过渡中，提供了 6 个 class 来切换：
	定义进入过渡的开始状态。在元素被插入之前生效，在元素被插入之后的下一帧移除。定义进入过渡生效时的状态。2.1.8版及以上 定义进入过渡的结束状态。定义离开过渡的开始状态。在离开过渡被触发时立刻生效，下一帧被移除。定义离开过渡生效时的状态。2.1.8版及以上 定义离开过渡的结束状态。##16.1.2CSS过渡
	通常我们使用css过渡来实现效果。
##16.1.3CSS动画
	CSS 动画用法类似 CSS 过渡，但是在动画中 v-enter 类名在节点插入 DOM 后不会立即删除，而是在 animationend 事件触发时删除。
#16.3初始渲染的过渡
	可以通过apper特性设置节点在初始渲染的过渡

17.vuejs混入
混入定义了一部分可复用的方法或者计算属性。混入对象可以包含任意组件选项。当组件使用混入对象时，所有混入对象的选项被混入该组件本身的选项。

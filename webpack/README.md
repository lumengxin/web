                 Webpack v3 快速入门
1.认识webpack
1.1 webpack是模块的捆绑器。主要目的是捆绑JavaScript文件以便在浏览器中使用，但它也能够转换，捆绑或打包任何资源或资产。
1.2 优势：再也不需要js标签，所有依赖库可通过npm安装；强大的插件系统，应付所有需求。

2.简单使用

自定义构建

      在线自定义构建
      用echarts/build/build.js脚本
      使用构建工具webpack,rollup

步骤：

  • 创建工程
  • 初始化：npm init; npm install echarts --save
  • 使用ES Module: import * as echarts from 'echarts'
  • 构建：node node_modules/echarts/build/build.js --min -i echarts.custom.js -o lib/echarts.custom.min.js
